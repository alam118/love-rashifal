# ❤️ Love Rashifal - प्रेम राशिफल

A modern, interactive web application providing daily love horoscope predictions with multiple features in Hindi & English.

## ✨ Features

### 🔮 6 Powerful Features

1. **🌟 Daily Horoscope** - आज का प्रेम राशिफल
2. **💑 Match Rashi** - राशि मिलान
3. **😊 Mood Score** - आज का मूड
4. **💕 Love Score** - प्रेम ऊर्जा
5. **💡 Today's Advice** - आज की सलाह
6. **🎲 Lucky Number** - भाग्यशाली नंबर

### 🎯 Key Highlights

- ✅ **Loading Screen** - Beautiful animated loading
- ✅ **Hindi & English** - Toggle between languages
- ✅ **Dark/Light Theme** - Eye-friendly themes
- ✅ **Share Feature** - Share results to WhatsApp, Telegram, Twitter, Copy
- ✅ **12 Zodiac Signs** - Complete astrology coverage
- ✅ **Modern UI** - Glassmorphism design
- ✅ **Responsive** - Works on all devices
- ✅ **Keyboard Shortcuts** - T=Theme, L=Language

## 🚀 Quick Start

1. Clone or download the files
2. Open `index.html` in your browser
3. Select a feature and your zodiac sign
4. View and share your results!
